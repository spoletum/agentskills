# AgentSkills

A comprehensive collection of agents, skills, and commands for AI-assisted development and automation workflows.

## Project Structure

```
agentskills/
├── agents/           # AI agent definitions and configurations
├── skills/           # Reusable skill definitions
├── commands/         # Command definitions for agent execution
├── templates/        # Project templates and boilerplates
├── docs/            # Documentation and guides
└── tests/           # Test suites for agents and skills
```

## Getting Started

1. Browse the `agents/` directory for available agents
2. Check `skills/` for specific capabilities you can use
3. Use `commands/` for pre-defined command templates
4. Reference `docs/` for detailed usage instructions

## Adding New Agents/Skills

1. Create your agent/skill file in the appropriate directory
2. Follow the existing naming conventions
3. Add documentation in the `docs/` folder
4. Include tests where applicable
